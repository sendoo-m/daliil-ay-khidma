## 📦 الملفات الجديدة

### 1. `apps/core/middleware.py`
**Middleware للغة:**
```python
class AdminEnglishMiddleware:
    """
    يجبر /admin على استخدام الإنجليزية
    باقي الموقع يستخدم العربية
    """
```

### 2. `config/settings/base.py`
**تحديثات الإعدادات:**
- إضافة `AdminEnglishMiddleware` لـ `MIDDLEWARE`
- إضافة `LOCALE_PATHS`
- `LANGUAGE_CODE = 'ar'` (العربية افتراضية)

### 3. `templates/base.html`
**قالب عصري بالعربية:**
- ✅ Bootstrap 5 RTL
- ✅ Font Awesome 6
- ✅ Google Fonts (Cairo)
- ✅ تصميم Gradient عصري
- ✅ Smooth animations
- ✅ رسائل ذكية (auto-dismiss)
- ✅ Scroll to top button
- ✅ Responsive design

**دليل شامل:**
