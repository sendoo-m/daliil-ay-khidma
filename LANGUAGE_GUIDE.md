# 🌍 دليل اللغات | Language Guide

## نظرة عامة | Overview

هذا الدليل يوفر مرجعاً شاملاً لجميع النصوص والترجمات المستخدمة في نظام دليل أي خدمة.

This guide provides a comprehensive reference for all texts and translations used in the Daliil Ay Khidma system.

---

## 📝 مبادئ الترجمة | Translation Principles

### 1. **الوضوح | Clarity**
- استخدم لغة واضحة ومباشرة
- Use clear and direct language
- تجنب المصطلحات المعقدة عندما يكون ذلك ممكناً
- Avoid complex terminology when possible

### 2. **الاتساق | Consistency**
- استخدم نفس المصطلحات في جميع أنحاء النظام
- Use the same terms throughout the system
- حافظ على نمط موحد للصياغة
- Maintain a uniform writing style

### 3. **الدقة | Accuracy**
- اترجم المعنى وليس فقط الكلمات
- Translate the meaning, not just words
- تأكد من دقة المصطلحات التقنية
- Ensure technical terms are accurate

### 4. **التوافق الثقافي | Cultural Compatibility**
- استخدم عبارات مناسبة للثقافة المحلية
- Use phrases appropriate for local culture
- راعِ العادات والتقاليد
- Respect customs and traditions

## 🖊️ أفضل الممارسات | Best Practices

### 1. **استخدام gettext_lazy**
```python
from django.utils.translation import gettext_lazy as _

class MyModel(models.Model):
    name = models.CharField(
        max_length=100,
        verbose_name=_('Name'),
        help_text=_('Enter the name')
    )
```

### 2. **استخدام ملفات الترجمة**
```bash
# إنشاء ملف الترجمة
python manage.py makemessages -l ar

# ترجمة في django.po
# locale/ar/LC_MESSAGES/django.po

# تجميع الترجمات
python manage.py compilemessages
```

### 3. **في القوالب (Templates)**
```django
{% load i18n %}

<h1>{% trans "Welcome to Daliil Ay Khidma" %}</h1>

{% blocktrans %}
This is a comprehensive service directory.
{% endblocktrans %}
```

### 4. **في الـ Views**
```python
from django.utils.translation import gettext as _

def my_view(request):
    messages.success(request, _('Operation completed successfully'))
```

## 📝 إرشادات الكتابة | Writing Guidelines

### للغة العربية | For Arabic

1. **استخدم العربية الفصحى المبسّطة**
   - تجنب العامية المحلية
   - استخدم لغة واضحة ومفهومة للجميع

2. **التشكيل**
   - ضع تشكيلاً فقط عند الضرورة
   - اترك النصوص البسيطة بدون تشكيل

3. **الأرقام**
   - استخدم الأرقام العربية في النصوص
   - الأرقام الإنجليزية مقبولة في الأرقام الكبيرة

### For English

1. **Use Simple English**
   - Avoid jargon when possible
   - Use clear, direct language

2. **Sentence Case**
   - Use sentence case for labels
   - Title Case for headings only

3. **Numbers**
   - Use Western Arabic numerals (0-9)
   - Format large numbers with commas
